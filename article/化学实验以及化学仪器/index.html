化学实验以及化学仪器

化学仪器（概论）

反应容器

可直接加热

  • 试管
  • 蒸发皿
  • 燃烧匙

可以间接加热的

加热这些仪器需要垫上石棉网，防止仪器炸裂（受热不均匀）

  • 烧杯
  • 烧瓶
  • 锥形瓶

存放仪器

  • 广口瓶（固体）
  • 细口瓶（液体）
  • 滴瓶（少量液体）
  • 集气瓶（存放气体，也可以作为气体的反应容器，但不能加热）

加热仪器

  • 酒精灯
  • 酒精喷灯

测量仪器

  • 托盘天平（重量）
  • 量筒（体积）

取用仪器

  • 镊子（块状药品或较大的金属颗粒）
  • 药匙（粉末状或小颗粒药品
  • 胶头滴管（少量液体）

固定仪器

  • 试管夹
  • 坩埚钳
  • 铁架台带铁夹、铁圈）

其他

石棉网、玻璃棒、试管刷、试管架、水槽、表面皿、长颈漏斗、分液漏斗、三脚架、研钵、点滴板

玻璃棒：搅拌，导流。

药品的取用

一：三不规则

不闻，不碰，不尝。
不能用手接触药品，不能尝任何药品的味道，更不得把鼻孔凑到容器口闻药品气味。

二：最少量原则

应严格按规定的用量取药品；若无说明用量，一般应按最少量取用：液体为1~2mL，固体只需盖满试管底部即可。

三：环保原则

用剩的药品不能放回原瓶，不要随意丢弃，不能拿出实验室，要放入指定容器内。

四：固体药品的取用

（1）密度较大或大颗粒固体的取用
主要仪器：镊子
步骤：一横，二放，三慢竖。
（2）粉末状固体药品的取用：
主要仪器：药匙，纸槽。
步骤：一斜，二送，三直立。
（3）固体药品的称量
主要仪器：托盘天平。
1. 托盘要放置在水平桌面上，调平，游码归零。
2. 药品放左边，砝码放右边。
3. 腐蚀性药品放在烧杯里。

五：液体药品的取用

（1）倾倒
1。标签向着手心。防止药品腐蚀标签。
2。瓶塞倒着放。防止瓶塞沾染其他物质污染药品。
3。瓶口贴着试管口。防止倒出。
（2）滴管取用
1.取液后应保持橡胶帽在上，不能平放和倒放，防止腐蚀橡胶帽。
2. 不要把滴管放在实验台或其他地方，以免沾污滴管。
3. 用过的滴管要立即用清水冲洗干净（滴瓶上的滴管不要用水冲洗），以备再用。 严禁用未经清洗的滴管再吸取别的试剂。
4. 胶头滴管不能搅拌。
5. 取用时不能挤入空气。
6. 滴管不能伸入容器，应该悬空垂直。
（3）液体药品的量取
主要仪器：量筒和胶头滴管。
1在接近所要量取的刻度时改用滴管。
2. 视线要与液体的凹液面的最低处保持水平
3. 俯视比实际值大
4. 仰视比实际小。

物质的加热

酒精灯

  1. 添加酒精用漏斗
    2。不超过酒精灯容积的三分之二，不少于四分之一。
  2. 用灯盖该灭酒精灯。
  3. 不能用酒精灯引燃酒精灯，用火柴。
  4. 不能往点燃的酒精灯加酒精。
  5. 着火用湿抹布盖住。
  6. 酒精灯火焰分为内外，焰心三层，外面温度最高，最内层温度最低，用最外层加热。

给试管中物质加热的注意事项

  1. 加热前应保证试管外壁的干燥。
  2. 加热前应该先预热，预热以后再用外焰对着放有药品的地方加热。
  3. 固体加热，平铺试管底部，略向下倾斜。
  4. 液体加热，不超过试管的三分之一，略向上倾斜。
  5. 试管口不对人。
  6. 加热后，试管不能立刻接触冷水。
  7. 夹持试管要夹在管口三分之一处。

仪器的连接

  1. 把玻璃管插入单孔橡胶塞。
  2. 把玻璃款和胶皮管连接。
  3. 把单孔橡胶塞塞到仪器上。
  4. 管口沾水，转动慢插。

检查气密性

连接好装置，将导管的一端放入盛水的容器中，用手掌紧贴试管或烧瓶外壁 ，如果导管口有气泡冒出，证明装置的气密性良好。

仪器的洗涤标准

仪器内壁附着的水既不聚成水滴，也不成股流下

仪器的组装与拆卸顺序

（1）组装仪器一般按从下而上、从左到右的顺序进行。
（2）拆卸仪器时，一般先拆开各仪器间的连接导管，然后由右到左、由上到下依次拆卸。
